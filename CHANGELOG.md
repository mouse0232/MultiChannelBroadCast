# 更新日志 - 2025年11月9日

## 🎉 重要更新

### 问题修复

#### 1. ✅ 图片路径问题已修复
**问题**: 图片URL前缀重复 `/static/https://...`  
**原因**: `STATIC_PROXY` 默认值设置为 `/static/`  
**解决**: 将默认值改为空字符串,如需代理请在 `.env` 中配置

**配置方法**:
```env
# 如果需要图片代理,设置此项
STATIC_PROXY=https://your-proxy.com/
```

---

#### 2. ✅ 分页优化
**改进**: 每页限制显示 20 条内容  
**之前**: 显示所有内容,页面过长  
**现在**: 
- 每页最多 20 条
- 显示加载进度 "显示 20 / 100 条内容"
- 更流畅的浏览体验

#### 3. ✅ 多频道独立页面

**页面结构**:

**导航栏特性**:

- 🎨 现代化圆角设计

---

#### 4. ✅ 自定义站点图标
**新功能**: 可以设置独立的站点头像

**配置方法**:
```env
# 方法1: 使用在线图片
SITE_AVATAR=https://your-domain.com/avatar.png

# 方法2: 使用本地图片(放在 public/ 目录)
SITE_AVATAR=/avatar.png
```

**优先级**: `SITE_AVATAR` > 频道头像

---

#### 5. ✅ 频道显示名称自定义
**新功能**: 可以为频道设置中文显示名

**配置方法**:
```env

```

## 🚀 使用说明

### 1. 更新环境变量

编辑 `.env` 文件:

```bash
# 移除 STATIC_PROXY 或设为空(如不需要代理)
STATIC_PROXY=

# 添加自定义头像(可选)
SITE_AVATAR=/avatar.png

# 添加频道显示名称(可选)
CHANNEL_NAMES=channel1:显示名1;channel2:显示名2
```

### 2. 添加站点头像

如果使用本地图片:
```bash
# 将你的头像图片放到 public 目录
copy your-avatar.png d:\Documents\broadcast\MultiChannelBroadcast\public\avatar.png
```

### 3. 重启开发服务器

```powershell
# 停止当前服务器 (Ctrl+C)
# 重新启动
pnpm dev
```

## 🎯 路由结构

```
首页(聚合):
  GET /                     → 所有频道最新20条
  GET /before/{id}          → 所有频道更早内容
  GET /after/{id}           → 所有频道更新内容

单频道:
  GET /channel/{name}       → 单频道最新20条
  GET /channel/{name}/before/{id}  → 单频道更早
  GET /channel/{name}/after/{id}   → 单频道更新

文章详情:
  GET /posts/{id}           → 单篇文章
```

---

## 💡 最佳实践

1. **频道数量**: 建议 3-5 个,不超过 7 个
2. **显示名称**: 使用简短易记的中文名
3. **站点头像**: 使用 256x256 或更大的正方形图片
4. **分页**: 保持默认 20 条/页,体验最佳
