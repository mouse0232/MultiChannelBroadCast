# Telegram 评论功能使用说明

## 📌 快速启用

### Cloudflare Pages 部署

1. 登录 Cloudflare Dashboard
2. 进入项目 → Settings → Environment variables
3. 添加变量:
   - Variable name: `COMMENTS`
   - Value: `true`
   - Environments: 勾选 Production 和 Preview
4. 保存后重新部署

### 本地开发

在 `.env` 文件中添加:
```env
COMMENTS=true
```

## ⚠️ 常见问题

### 错误: ERR_BLOCKED_BY_CLIENT

**原因**: 浏览器的广告拦截器(如 AdBlock、uBlock Origin)拦截了 Telegram widget 脚本。

**解决方法**:

1. **临时禁用广告拦截器**
   - 点击浏览器工具栏的广告拦截器图标
   - 选择"在此站点上禁用"

2. **添加到白名单**(推荐)
   
   **uBlock Origin**:
   - 点击扩展图标 → 点击大电源按钮(变灰)
   - 或进入设置 → 白名单 → 添加域名
   
   **AdBlock**:
   - 点击扩展图标 → "不在此页面上运行"
   - 或设置 → 自定义 → 添加域名到白名单

3. **直接访问 Telegram**
   - 页面会显示 Telegram 链接
   - 点击链接在 Telegram 中查看评论

## 🔍 检查评论是否启用

打开浏览器开发者工具(F12),在 Console 中输入:
```javascript
document.querySelector('.comments')
```
- 如果返回元素对象: 评论功能已启用
- 如果返回 `null`: 评论功能未启用,检查环境变量

## 📋 必要条件

- ✅ 环境变量 `COMMENTS=true`
- ✅ Telegram 频道开启了讨论组功能
- ✅ 浏览器允许加载 telegram.org 的脚本

## 💡 提示

- 评论数据存储在 Telegram,不占用服务器资源
- 首次加载需要 1-3 秒
- 用户需要登录 Telegram 才能发表评论
- 如果看到黄色提示框,说明脚本被拦截了
